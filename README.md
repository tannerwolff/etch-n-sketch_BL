# etch-n-sketch_bl

Creating an etch-n-sketch with HTML, CSS, and JS

Etch-N-Sketch plans/pseudocode

1. Create default 16x16 grid dynamically with empty divs
2. Add event listeners and a grid-item class to each for modifications
3. Use Flexbox to create the 16x16 grid using wrap and custom widths
4. Use event listeners to change background colors of hovered divs
5. Add button and functionality to allow user to pick size(under 100 x 100)
6. Button should clear old grid and add the new user specified one
7. Add color randomizer to change the colors if desired
